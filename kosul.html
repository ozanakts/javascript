JavaScript Koşul İfadelleri

if İfadesi

If ifadesi, eğer verilen koşul doğruysa true, değilse false değerini verir. Aşağıda if'in kullanımını görüyorsunuz.

if (koşul) {
Koşul sağlanıyorsa yapılacaklar
}

İf...else İfadesi

If ifadesiyle bir veye birkaç koşul belirttiğimizde, oluşabilecek bir durum da bu koşul veya koşullardan hiçbirinin gerçeklenmemesidir.

if (koşul) {
Koşul sağlanıyorsa yapılacaklar
}
else {
Bunun dışındaki her durumda yapılacak iş
}
